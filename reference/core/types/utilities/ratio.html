ratio¶

struct units::ratio¶

Provides compile-time rational arithmetic support.

This class is really similar to std::ratio but gets an additional Exp template parameter that defines the exponent of the ratio. Another important difference is the fact that the objects of that class are used as class NTTPs rather then a type template parameter kind.
