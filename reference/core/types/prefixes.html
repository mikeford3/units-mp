Prefixes¶

struct prefix_family¶

The base for all prefix families.

Every prefix family should inherit from this type to satisfy PrefixFamily concept.

Subclassed by units::isq::iec80000::binary_prefix, units::isq::si::prefix, units::no_prefix

struct no_prefix : public units::prefix_family¶

No prefix possible for the unit.

This is a special prefix type tag specifying that the unit can not be scaled with any kind of the prefix.
