Metafunctions¶

template<Dimension D>
using units::dimension_unit = decltype(detail::default_unit<D>())¶

Returns a ‘default’ unit of the dimension.

Depending on the dimension type it returns a base unit (for base dimensions) or a coherent unit (in case of derived dimensions).

Template Parameters

  • D: Dimension type to get the unit from.
